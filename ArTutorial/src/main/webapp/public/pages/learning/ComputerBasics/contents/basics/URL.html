Frequently Asked Questions


What is the difference between statically typed and dynamically typed language?





	
Statically typed languagesDynamically typed language

	
The language where the type of variables are known during compilation.The language where the type of variables is known during run time.

	
Examples are Java, C, C++Examples are Perl, Ruby, Python






What is the difference between a compiled and an interpreted language?





	
Compiled languagesInterpreted language

	
The language where the human-readable code is converted to machine code that the computer can run natively.The language that takes the human-readable code and turns it into something else before making it something the machine can read.

	
It is faster in execution.It is comparatively slower in execution.

	
The errors can be detected earlier during compilation phase.The errors can be detected only during run time.

	
Examples are Java, C, C++Examples are Python, Javascript
