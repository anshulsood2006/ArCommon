Basic Calculas

Features

Derivative

  • The derivative of a function of a real variable is the measures of its sensitivity to change its value (output value) with respect to a change in its argument (input value).
  • It is calculated for a function of single variable.
  • It is a scalar-valued function.
  • Example: the derivative of the position of a moving object with respect to time is the object's velocity: this measures how quickly the position of the object changes when time advances.
  • The derivative of y with respect to x is written as dy/dx

Gradient

  • The gradient of a function of a real variable is the measures of its sensitivity to change its values (output value) with respect to a changes in its arguments (input value).
  • It is calculated for a function of multiple variables.
  • It is a vector-valued function.

Partial Derivative

  • Partial Derivative of a function of several variables is its derivative with respect to one of those variables, with the others held constant
  • For example for a function f(x,y,z) partial derivative with respect to x is denoted as δf/ δx

Transpose

The transpose of a matrix is a new matrix whose rows are the columns of the original matrix.
