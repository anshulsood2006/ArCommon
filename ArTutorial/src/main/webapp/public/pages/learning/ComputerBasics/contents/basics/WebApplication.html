Web Application

  • An application that can be accessed from anywhere using a browser is called a web application.
  • Web Server is a software that listens to user requests on a specific port e.g. apache tomcat, jboss.
  • Web Client is a software that is used to send a user request to web server and get back the response from web server e.g. browser
  • Web server and web client interact with each other using HTTP protocol that runs on top of TCP/IP communication protocol.
