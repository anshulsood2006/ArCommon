MySQL

Features

  • Install MySQL Server
    • Add the MySQL repository to yum repository using command yum install mysql80-community-release-el8-1.noarch.rpm
    • Enable the repository using command yum config-manager --enable mysql80-community
    • Disable MySQL using command yum module disable mysql
    • Install MySQL using command yum install mysql-community-server
  • Get the version of MySQL service in linux using command mysqld --version
  • Get the status of MySQL service in linux using command service mysqld status
  • Start MySQL service in linux using command service mysqld start
  • Stop MySQL service in MySQL using command service mysqld stop
  • Change root password of mysql after fresh installation using command below steps
    • Get the temporary password using command grep 'temporary password' /var/log/mysqld.log
    • Change the password using command mysql_secure_installation
  • Start MySQL using command mysql -u root -p
