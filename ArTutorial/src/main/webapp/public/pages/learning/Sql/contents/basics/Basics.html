How to install postgres in linux?


    
	
  • Configure yum repository for postgres using command: yum -y install https://download.postgresql.org/pub/repos/yum/reporpms/EL-8-x86_64/pgdg-redhat-repo-latest.noarch.rpm
    • 
	
  • Disable the module "postgresql" from the official appstream distribution using command: yum -qy module disable postgresql
    •  
	
  • Install postgres using command: yum install postgresql12-server postgresql12  postgresql12-contrib
    • 
	
  • Initialize database cluster using command: /usr/pgsql-12/bin/postgresql-12-setup initdb
    • 
	
  • Enable automatic restart on boot using command: systemctl enable --now postgresql-12
    • 
	
  • Start the service using command: systemctl status postgresql-12
    • 
	
  • Enable database access for external machines edit file /var/lib/pgsql/12/data/postgresql.conf and add listen_addresses = '*'
    • 
	
  • Restart the database server using command: systemctl restart postgresql-12
    • 
	
  • Confirm the port using command: netstat -antup | grep 5432
    • 
	
  • Login as postgres user using command: su -l postgres
    • 
	
  • Change default password using command: \password
    • 
	
  • To find the location of pg_hba.conf file use command SHOW hba_file;
    • 
	
  • To prompt password during each login to database 
		
      
			
    • Edit postgresql.conf and add md5 to line having content local  all  all md5
      • 
			
    • Reload the configuration using commmand: pg_ctl reload -D /var/lib/pgsql/12/data/
      • 
			
    • Restart the database server using command: systemctl restart postgresql-12
      • 
			
    • Verify the above using command psql -qAt -c "show hba_file" | xargs grep -v -E '^[[:space:]]*#'
      •
