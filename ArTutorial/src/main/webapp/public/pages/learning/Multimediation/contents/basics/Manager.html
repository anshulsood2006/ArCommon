Multimediation Manager

Configuration

  • When a manager application starts it reads the web.xml file inside WEB-INF.
  • There are 3 listeners defined for the manager
    • org.springframework.web.context.ContextLoaderListener This listener initializes all the beans defined in the xml files which are referred in the context-param for parameter contextConfigLocation
      • applicationContext.xml Beans are
        • com.ericsson.cm.manager.beanfactory.BeanManager
        • com.ericsson.cm.manager.filesystem.FileSystemManager
        • com.ericsson.cm.manager.filesystem.RemoteFileExecutor
        • com.ericsson.cm.manager.usermanagement.model.UserManagerImpl
        • com.ericsson.cm.manager.configurationmanager.ConfigManagerImpl
          • There are some beans which are hibernate Dao objects e.g. HibernateAlarmViaEmailConfigDAO.
          • These Dao objects are using hibernateTemplate instead of session factory which is not recommended.
      • dataManagementContext.xml
      • applicationContextTracer.xml
      • FileMediationDataMgmtContext.xml
    • org.springframework.web.context.request.RequestContextListener
    • com.ericsson.cm.manager.sessionmanager.core.model.SessionManagerExecutorServiceListener
