Kubernetes

Features

  • Kubernetes is a container orchestration platform developed by Google. It is an enhanced version of Borg tool which was used to manage both long running processes and batch jobs.
  • It was donated to Cloud Native Computing Foundation (CNCF) by Google and thus became open source.
  • It deals in automated deployment, scaling, running and monitoring of applications.
  • It acts as a control loop as it compares the desired state to the actual state, and if they aren't the same, takes steps to correct it.

Components of Kubernetes

  • Master Control Panel: The component that performs orchestration. It has below sub components
    • Etcd: For storage of configuration information.
    • API Server: For communication between the components.
    • Schedular: For deciding the nodes where pods should run.
    • Controller Manager: For checking current state with desired state.
  • Pods: The component that hold containers, most of the times a pod has one container.
  • Nodes: The component where pods get scheduled. The applications actually run on the nodes.
