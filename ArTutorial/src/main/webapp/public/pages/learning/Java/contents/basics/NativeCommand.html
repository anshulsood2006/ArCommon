Java Native Commands

Features

The native means the code that is written in a language other than java

Examples

Creating a native method in JAVA

  • Create a java class having native method
  • Select the class
  • Go to Run >> External Tool >. External Tool Configurations
  • Create a new configuration with some name say ""Native"
  • Select location as C:\Program Files\Java\jdk1.8.0_151\bin\javah.exe
  • Select the working directory as ${workspace_loc:/ArCommon/target/classes}
  • Select arguments as -jni com.arsoft.projects.common.learning.natives.ArNativeClass
  • Click run
  • This will return a file named e.g. com.arsoft.projects.common.learning.natives.ArNativeClass.h
  • Add the implementation for the method in C or C ++
  • Compile the library to .dll or .so file
