Inheritance

Features

  • In case of inheritance, the constructor of base class with no argument is always called whenever any constructor of derived class is called.
  • In case of inheritance, the method of the class whose constructor is invoked is called.
  • In Java there is only call by value.

Frequently Asked Questions

What is the output of below program:

public class ClassA {

	public ClassA() {
		System.out.println("constructor of Class A is called");
	}
	
}

public class ClassB extends ClassA {
	
	public ClassB() {
		System.out.println("constructor of Class B is called");
	}
	
	public ClassB(String B) {
		System.out.println("constructor of Class B is called with argument "+B);
	}
	
	public static void main(String[] args) {
		ClassA clazz = new ClassB();
	}
}

constructor of Class A is called
constructor of Class B is called

What is the output of below program:

public class ClassA {
	
	public void method() {
		System.out.println("method of Class A is called");
	}
	
}

public class ClassB extends ClassA {
	
	public void method() {
		System.out.println("method of Class B is called");
	}
	
	public static void main(String[] args) {
		ClassB classb = new ClassB();
		classb.method();
		ClassA classa = new ClassB();
		classa.method();
		classa = new ClassA();
		classa.method();
	}
}

method of Class B is called
method of Class B is called
method of Class A is called
