Static

Features

  • A static method/variables belong to the class rather than object of a class.
  • A Static variable is associated with class and not to object and hence it is not serialized.
  • A static method can be invoked without the need for creating an instance of a class.
  • A static method can access static data member and can change the value of it.

Frequently Asked Questions

Why we can not use this or super keyword in a static context?

This keyword refers to the current object, while the super keyword refers to immediate parent class object. So both these keywords belong to instance of the class. Whereas, static method belongs to the class than instance of the class. And hence we can not use this or super keyword in a static context.
