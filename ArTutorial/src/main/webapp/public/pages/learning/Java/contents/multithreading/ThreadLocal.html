Thread Local

    https://javarevisited.blogspot.com/2012/05/how-to-use-threadlocal-in-java-benefits.html
  • It is a variable in Java that can be read/written by same thread only. No two threads can see each other's thread local variables.
  • It is created using ThreadLocal class.
  • Methods are
    • public void set(T value)
    • public Object get()
  • These variables provide thread safety by providing explicit copy of objects to each thread. Since the objects are different for each thread there is no need to synchronize these objects.
