Functional Interface

Features

  • An interface having one and only one abstract method is called a functional interface
  • A functional interface can have any number of default concrete method.
  • A functional interface should be annotated with annotation @FunctionalInterface .
  • An instance of functional interface can be created using lambda expression.
  • A lambda expression can be used to avoid bulky anonymous implementation.

Examples

java.lang.Runnable having one abstract method run()
