Comparator & Comparable

Features

ComparableComparator
It is used to sort elements of a user defined collection on the basis of SINGLE class variable using natural sorting order. It is used to sort elements of a user defined collection on the basis of ONE or MORE class variables using user defined sorting order.
Sorting logic is in the same class whose objects are being sorted. So this is called natural ordering of objects Sorting logic is in the different class whose objects are being sorted. So we can write different logics based on different attributes of objects
The class whose objects needs to be sorted must implement Comparable interface The class whose objects needs to be sorted need not implement Comparator interface. Instead new class is written that implements the Comparator interface
It uses compareTo(ArClass clazz) method to sort the elements. It uses compare(ArClass obj1, ArClass obj2) method to sort the elements.
It is found inside java.lang package. It is found inside java.util package.
Elements can be sorted using Collection.sort(list) method. Elements can be sorted using Collection.sort(list, comparator) method.
It compares this object to the given object.
  • Returns -1 if this < object.
  • Returns 0 if this == object.
  • Returns 1 if this > object.
 It compares two given objects.
  • Returns -1 if object1 < object2.
  • Returns 0 if object1 == object2.
  • Returns 1 if object1 > object2.

Examples

Comparable

public class ArStudent implements Comparable {

	private int age;
	private int rollNumber;
	private String name;
	
	public ArStudent(String name, int age, int rollNumber) {
		this.name = name;
		this.age = age;
		this.rollNumber = rollNumber;
	}
	
	public int getAge() {
		return age;
	}

	public void setAge(int age) {
		this.age = age;
	}

	public int getRollNumber() {
		return rollNumber;
	}

	public void setRollNumber(int rollNumber) {
		this.rollNumber = rollNumber;
	}

	public String getName() {
		return name;
	}

	public void setName(String name) {
		this.name = name;
	}

	@Override
	public int compareTo(ArStudent student) {
		if (this.name == null || student.name == null){
			return 0;
		}
		return this.name.compareTo(student.name);
	}
	
	public String toString(){
		return this.name;
	}
	
	public static void main(String[] args) {
		ArStudent a = new ArStudent(null, 0, 0);
		ArStudent b = new ArStudent("A", 10, 1);
		ArStudent c = new ArStudent("z", 10, 2);
		ArStudent d = new ArStudent("P", 11, 3);
		ArStudent e = new ArStudent("Z", 9, 4);
		List list = new ArrayList<>();
		list.add(a);
		list.add(b);
		list.add(c);
		list.add(d);
		list.add(e);
		System.out.println("Before: "+list);
		Collections.sort(list);
		System.out.println("After: "+list);
	}

}
