Static Method

Features

  • From java 1.8 onwards a java interface can have a method with keyword static.
  • The implementation needs to be provided in the interface itself.
  • This method can be called using the interface name only in the child class
  • The static method can not be overridden in the child classes as being a static method, it belongs to interface only. However the same static method can be created in the child classes which will eventually belong to that class only.
