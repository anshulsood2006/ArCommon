Frequently Asked Questions


Why I am getting a null mock object when I am using annotation @Mock to mock an instance of a class?


Make sure that the test class is annotated with annotation @RunWith(MockitoJUnitRunner.class)


@RunWith(MockitoJUnitRunner.class)
public class ArDictionaryTest {
	@Mock
	ArDictionary arDictionary;
	
	@Test
	public void addTest(){
		boolean added = arDictionary.add("Anshul", "Sun");
		assertTrue(added);
	}
}
