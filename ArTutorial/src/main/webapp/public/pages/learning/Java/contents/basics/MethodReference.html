Method Reference

Features

  • It is used to refer to static or non static methods or a constructor of a class.
  • It is a shorthand syntax for a lambda expression that executes just ONE method. Syntax is: Object :: methodName
  • It is used to pass just a reference of a method to another method instead of passing the complete object.
  • It is of three types:
    1. Static Method Reference
    2. Non Static Method Reference
    3. Constructor Reference
