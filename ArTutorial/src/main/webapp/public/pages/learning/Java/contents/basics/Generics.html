Generics

Features

  • It was introduced in Java 1.5
  • It provided the compile time type safety to collections and helps to eliminate unnecessary type casting. e.g.
    • List<Integer> myIntList = new LinkedList<Integer>(); will create a list of Integer objects.
    • List<T> myIntList = new LinkedList<T>(); will create a list of any kind of objects.
  • The abbreviations used most commonly are:
    • E for element
    • T for type
    • K for key
    • V for value
    • N for number
