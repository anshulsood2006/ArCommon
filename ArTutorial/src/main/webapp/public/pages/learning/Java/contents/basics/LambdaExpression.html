Lambda Expressions

Features

  • A lambda expression is a function that can be created without belonging to any class.
  • It was introduced on Java 8
  • It is used to implement functional interfaces in java. Functional interface is an interface having one and only one abstract method.
  • It is also used in functional programming using Java Streams API.
  • It is used to avoid boilerplate code.
  • The syntax is (Argument-list) Arrow-token -> {body}
  • It gives a compile time error "The target type of this expression must be a functional interface" if used with a non functional interface.

Examples

Lambda expression to implement method having no arguments

public interface ILambdaExpressionTestInterfaceNoArgument {
	public String hello();
}
public class LambdaExpressionTestClass{
	public static void main(String[] args) {
		ILambdaExpressionTestInterfaceNoArgument noLambda = () -> {return "Hello";};
		System.out.println(noLambda.hello());
	}
}

Lambda expression to implement method having one argument

public interface ILambdaExpressionTestInterfaceOneArgument {
	public String hello(String firstName);
}
public class LambdaExpressionTestClass{
	public static void main(String[] args) {
		ILambdaExpressionTestInterfaceOneArgument oneLambda = (firstName) -> {return "Hello " +firstName;};
		System.out.println(oneLambda.hello("Anshul"));
	}
}

Lambda expression to implement method having two arguments

public interface ILambdaExpressionTestInterfaceTwoArgument {
	public String hello(String firstName, String lastName);
}
public class LambdaExpressionTestClass{
	public static void main(String[] args) {
		ILambdaExpressionTestInterfaceTwoArgument twoLambda = (firstName, secondName) -> {return "Hello " +firstName +" "+secondName;};
		System.out.println(twoLambda.hello("Anshul","Sood"));
	}
}

Frequently Asked Questions

What will happen if we try to use lambda expression for an interface which is not a functional interface?

This will give a compile time error "The target type of this expression must be a functional interface"
