Lambda Expressions

Features

  • A lambda expression is a function that can be created without belonging to any class.
  • It was introduced on Java 8
  • It is used to implement functional interfaces in java. Functional interface is an interface having one and only one abstract method.
  • It is also used in functional programming using Java Streams API.
  • It is used to avoid boilerplate code.
  • The syntax is (Argument-list) Arrow-token -> {body}
