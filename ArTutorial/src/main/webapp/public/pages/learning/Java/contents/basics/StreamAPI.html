Frequently Asked Questions


How can we create a stream of a collection object


The default method stream() present inside collection interface can be used to return a stream of elements e.g. 

	Stream<String> stream = list.stream();
	//list is a list of String object 



How can we create a stream from arrays


This can be done in two ways:

    
	
  • Stream<String> stream = Stream.of("a","b", "c");
    • 
	
  • String[] arr = new String[]{"a", "b", "c"};
    
		Stream stream = Arrays.stream(arr);
