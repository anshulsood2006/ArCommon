SSL

Features

  • SSL (Secure Sockets Layer) is a protocol that allows your browser to create a safe connection between itself and a website or server.
    • For a client, it verifies that the legitimate and appropriate company owns the website that's being accessed.
    • For a server, it verifies that visitors are visiting the correct site and helps prevent other people from impersonating your company and website.
  • Keystore: It is used to store server's own identity/certificates. It contains the server's certificate, including its private key.
  • Truststore: It is used to store identity/certificates of other parties issued by CA
  • JSSE: Java Secure Socket Extension

Examples

Create a SSL connection between web server and the client.

  • Generate certificate:

    keytool -genkey -alias localhost -keyalg RSA -keypass password -storepass password -keystore server.keystore.jks

  • Convert to cer file:

    keytool -export -alias localhost -storepass password -file server.cer -keystore server.keystore.jks

  • Add the certificate to the trust store file:

    keytool -import -v -trustcacerts -alias localhost -file server.cer -keystore server.truststore.jks -keypass password -storepass password

  • Verify if the certificate was successfully added into the keystore.

    keytool -list -v -keystore server.keystore.jks

  • Validate if the certificate was successfully added into the trust store.

    keytool -list -v -keystore server.truststore.jks

  • Delete a certificate file:

    keytool -delete -alias sb6270x1769 -keystore server.keystore.jks

Create certificates using OpenSSL.

  • Download OpenSSL from internet
  • Create a sample directory and go to that directory from command prompt
  • SET OPENSSL_CONF=path to openssl.cfg file
  • SET RANDFILE=.rnd
  • Generate root certificate key: genrsa -out ca.key 4096
  • Generate root certificate: req -new -x509 -days 1826 -key ca.key -out ca.crt
  • Generate intermediate certificate key: genrsa -out ia.key 4096
  • Generate intermediate certificate signing request: req -new -key ia.key -out ia.csr
  • Generate intermediate certificate: x509 -req -days 730 -in ia.csr -CA ca.crt -CAkey ca.key -set_serial 01 -out ia.crt
  • Package the files: pkcs12 -export -out ia.p12 -inkey ia.key -in ia.crt -chain -CAfile ca.crt
  • Generate certificate with only public key: pkcs12 -in ia.p12 -clcerts -nokeys -out ca-public.crt
