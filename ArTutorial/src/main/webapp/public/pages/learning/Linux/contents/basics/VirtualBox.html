Install linux on Virtual Box

Steps

  • Download and install virtal box from Oracle's site
  • Download the DVD iso file of Redhat from Redhat's site
  • Mount the DVD
  • Select workstation > Container management, Development Tools and Graphical Administration Tools under Software
  • Disable KDUMP to save memory
  • Select ethernet under network adapter
  • Create root password and a user
  • Begin the installation
  • Unmount the disc before reboot
  • Agree to the license and get the system registered
  • Update yum repository using sudo yum update
  • Reboot the system.
  • sudo yum groupinstall "Development Tools"
  • sudo yum install kernel-devel elfutils-libelf-devel
  • Mount the Guest Additions DVD iso and run it
