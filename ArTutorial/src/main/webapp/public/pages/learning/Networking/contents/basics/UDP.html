User Datagram Protocol (UDP)

Features

  • It is a connection less protocol.
  • It offers no reliability, flow-control, or error-recovery functions.
  • The packets can not be discarded before reaching their targets
  • It is the transport protocol for the below well-known application-layer protocols
    • Network File System (NFS-UDP ports 1021/1022)
    • Simple Network Management Protocol (SNMP-UDP ports 161/162)
    • Domain Name System (DNS-UDP port 53)
    • Trivial File Transfer Protocol (TFTP-UDP port 69).
