Kafka

Features

  • Kafka: is a distributed publish-subscribe messaging system that can handle high volume of data.
  • Zookeeper: In case of kafka cluster, there are various brokers to maintain load balance. These brokers are managed by zookeeper. In case of failure of any broker, zookeeper notifies the consumer and producer

Frequently Asked Questions


    
	
  • How to set up and use Kafka?
		
      
			
    • Download Kafka
      • 
			
    • Start zookeeper using below command from the installation directory
      
			./bin/zookeeper-server-start.sh config/zookeeper.properties
      • 
			
    • Start kafka server using below command from the installation directory
      
			./bin/kafka-server-start.sh config/server.properties
      • 
			
    • Create a topic using below command from the installation directory
      
			./bin/kafka-topics.sh --create --zookeeper localhost:2181 --replication-factor 1 --partitions 1 --topic Topic
      • 
			
    • Get list of all the topics using below command from the installation directory
      
			./bin/kafka-topics.sh --list --zookeeper localhost:2181
      • 
			
    • Delete a topic using below command from the installation directory
      
			./bin/kafka-topics.sh --zookeeper localhost:2181 --delete --topic Topic
      • 
			
    • Produce a message and add it to the topic using below command from the installation directory
      
			./bin/kafka-console-producer.sh --broker-list localhost:9092 --topic Topic
      • 
			
    • Consume a message from the topic using below command from the installation directory
      
			./bin/kafka-console-consumer.sh --bootstrap-server localhost:9092 --topic Topic --from-beginning
