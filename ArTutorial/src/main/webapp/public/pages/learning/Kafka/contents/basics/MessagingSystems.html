Messaging System (Message Broker)

  • A message system or a message broker is an application system which is responsible for transferring data from one application to another
  • There are two types of messaging patterns
    • Point to Point messages
      • Messages are persisted in a queue.
      • A particular message can be consumed by one consumer only.
    • Publish-Subscribe messages
      • Messages are persisted in a topic.
      • A particular message can be consumed by many consumers.
