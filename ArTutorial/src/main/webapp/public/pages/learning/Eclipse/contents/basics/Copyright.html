Add Copyright to Java Code

Features

  • Create a file License.txt with the copyright text and place it inside the base directory of the project.
  • Add the below plugin dependencies to the project 
    <plugin>
	<groupId>com.mycila.maven-license-plugin</groupId>
	<artifactId>maven-license-plugin</artifactId>
	<version>${maven-license-plugin.version}</version>
	<configuration>
		<header>${license.dir}/license.txt</header>
		<properties>
			<project>${project.name}</project>
			<founder>${project.organization.name}</founder>
			<year>${project.inceptionYear}</year>
			<website>${founder-website}</website>
		</properties>
		<includes>
			<include>src/main/java/**</include>
			<include>src/test/java/**</include>
		</includes>
	</configuration>
	<executions>
		<execution>
			<goals>
				<goal>format</goal>
			</goals>
			<phase>process-sources</phase>
		</execution>
	</executions>
</plugin>
  • Run maven command "mvn license:format"
