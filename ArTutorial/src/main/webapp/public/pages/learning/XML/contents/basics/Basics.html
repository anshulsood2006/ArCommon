XML

Features

  • XML viz. Extensible mark up language is a mark up language.
  • Two API's used for XML are
    • SAX (Simple API for XML): This is an event-driven XML parsing where we need to register callbacks for events of interest and then let the parser proceed through the document. This is useful when the documents are large or we have memory limitations, it parses the file as it reads it from disk and the entire file is never stored in memory.
    • DOM (Data Object Model): This is a World Wide Web Consortium recommendation wherein the entire file is read into memory and stored in a hierarchical (tree-based) form to represent all the features of an XML document.
