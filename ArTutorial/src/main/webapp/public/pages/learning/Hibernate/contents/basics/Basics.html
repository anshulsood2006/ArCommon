Hibernate

Features

Object Relational Mapping (ORM)

  • This open source persistent framework was created by Gavin King in 2001.
  • Mapping Java objects to relational database table is called ORM.
  • It is provided by various frameworks like
    • EJB
    • JDO
    • Hibernate
    • DAO etc.

Hibernate as an ORM framework

  • Provides XML file or annotations based configuration to map Java classes to relational database tables.
  • Provides api to store/retrieve data to and from database.
  • Does not require an application server.
