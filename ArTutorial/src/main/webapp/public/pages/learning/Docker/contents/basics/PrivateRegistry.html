Set up Private Docker Registry

Steps

  • Pull the registry image from docker hub using command: docker pull registry
  • Run the docker container in detached mode using command: docker run -d -p 5000:5000 --name localregistry registry
  • Verify the docker container using command: ps -a
  • Download image from private registry using command: docker pull localhost:5000/ubuntu-git
  • Tag the image to the tag of local registry using command: docker tag anshulsood2006/ubuntu-git localhost:5000/anshulsood2006/ubuntu-git
  • Push the tag to the local registry using command: docker push localhost:5000/anshulsood2006/ubuntu-git
