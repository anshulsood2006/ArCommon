Docker Compose

Installing docker compose on RHEL8

  • Execute command: curl -L "https://github.com/docker/compose/releases/download/1.25.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
  • Execute command: chmod +x /usr/local/bin/docker-compose

Features

  • Docker Compose is a tool used to define and run multi-container Docker applications.
  • All the application services are configured inside a compose file named docker-compose.yml
  • A single command docker-compose up is used to create and start all the services from the configuration.
  • The yaml file can be vealidating using command: docker-compose config
  • The services can be started using command: docker-compose up
  • The services can be stopped using command: docker-compose down

Examples

Sample docker-compose.yml

	version: "3"
	
	services:
	  myweb:
	    image: ubuntu
	  mydb:
	    image: redis



Frequently Asked Questions


How to resolve error "mapping values are not allowed here"


Make sure that there are space after each colon in yaml file. They are mandatory.
