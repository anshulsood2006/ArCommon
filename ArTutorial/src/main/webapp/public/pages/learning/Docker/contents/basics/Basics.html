Docker

Features

  • Docker works on the principle of Containerization
  • Docker is a container management service.
  • It lets the developers to easily develop the applications and ship them inside containers which can be deployed anywhere.
  • The docker containers can be deployed anywhere on physical machines or Virtual machines or even on cloud.
  • The docker containers are lightweight and easily scalable.

Components of Docker

  • Docker
  • Docker Engine
  • Docker Hub
  • Docker Compose

Installation of Docker on Windows

  • Download and run Docker.exe from docker hub.

Installation of Docker on RedHat

  • Add the docker repository to yum repository list using command yum config-manager --add-repo=https://download.docker.com/linux/centos/docker-ce.repo.
  • Get the list of available docker versions using command yum list docker-ce --showduplicates | sort -r
  • Install the latest version using command yum install docker-ce --nobest -y
  • Get the status of docker service using command service docker status
  • Start docker service using command service docker start
  • Stop docker service using command service docker stop
  • Run a docker image using command docker run {image}. This command first looks for docker image on local system. If image is not found locally, it downloads the same from docker hub.
