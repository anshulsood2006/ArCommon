Docker

Features

  • Docker is a container management service.
  • It lets the developers to easily develop the applications and ship them inside containers which can be deployed anywhere.
  • The docker containers can be deployed anywhere on physical machines or Virtual machines or even on cloud.
  • The docker containers are lightweight and easily scalable.

Components of Docker

  • Docker
  • Docker Engine
  • Docker Hub
  • Docker Compose

Installation of Docker (Windows)

  • Download and run Docker.exe from docker hub.
