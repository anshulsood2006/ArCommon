Frequently Asked Questions


How to resolve error "Error parsing {android-sdks}\system-images\android-23\android-tv\armeabi-v7a\devices.xml"?


Go to Windows >> Andriod SDK Manager and uninstall the fauly package from API 23.


How to resolve error "No system image installed for this target"?


Go to Windows >> Andriod SDK Manager and install the missing System Image packages.
