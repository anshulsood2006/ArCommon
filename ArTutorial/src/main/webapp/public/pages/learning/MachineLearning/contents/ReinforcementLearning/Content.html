Reinforcement Learning

Features

  • Q-Table:
    • It is simple lookup table where we calculate the maximum expected future rewards for action at each state.
    • The columns represent the actions whereas the rows represent the states.
    • E.g. for a mine game:
      • A robot can perform four actions viz. move up, down, left or right.
      • The state of a robot at any point of time can be any of five actions viz. start, nothing, power, mine, end
      • The Q-Table will be as shown below
        Action
        State        		UpRightDownLeft
        Start
        Nothing
        Power
        Mine
        End
      • Each Q-table score will be the maximum expected future reward that the robot will get if it takes that action at that state.
      • Q-Table data needs to be improved at each iteration.
