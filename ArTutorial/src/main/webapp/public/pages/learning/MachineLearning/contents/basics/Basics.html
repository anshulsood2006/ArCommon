Machine learning

  • The term "machine learning" was coined in 1959 by Arthur Lee Samuel
  • It is the field of study that gives computers the ability to learn without being explicitly programmed.
  • It deals with programming the computer systems to make them learn and improve with data and experience.
  • Learning means recognizing and understanding the input data and decisions based on the supplied data.
  • It is used in various fields such as:
    • Vision processing
    • Language processing
    • Forecasting
    • Pattern recognition
    • Games
    • Data mining
    • Expert systems
    • Robotics
  • Steps involved are:
    • Defining a Problem
    • Preparing Data
    • Evaluating Algorithms
    • Improving Results
    • Presenting Results
  • A computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with experience E.
  • The goal of machine learning is never to make "perfect" guesses, because it deals in domains where there is no such thing. The goal is to make guesses that are good enough to be useful.
  • Predictor Function
    • Mathematically for machine learning, the goal is to devise a mathematical function h(x) called predictor or hypothesis. Learning consists of using sophisticated mathematical algorithms to optimize this function so that, given input data x about a certain domain, it will accurately predict some interesting value h(x). Example:
      h(x) = θ0 + θ1 * x
    • θ0 and θ1 are called coefficients/parameters of the predictor function.
  • Cost Function or Loss Function
    • The wrongness measure in estimating the almost accurate value of predictor function h(x) is known as the cost function or loss function.Example:
      J(θ0, θ1) = 1/2m Σ (h(xt,i) - y)2 where i ranges from 1 to m
    • The ultimate goal of machine learning is to find the coefficients for predictor h(x) such that our cost function J (θ0, θ1) is as small as possible.The power of calculus is used to accomplish this.
    • The cost function when plotted against the values of θ0 and θ1 gives a bowl shaped plot.
      • Any random point on the bowl plot gives the cost of the current values of the coefficients
      • The bottom of the bowl is the cost of the best set of coefficients, the minimum of the function.
      • The goal is to continue to try different values for the coefficients, evaluate their cost and select new coefficients that have a slightly better (lower) cost. Repeating this process enough times will lead to the bottom of the bowl and you will know the values of the coefficients that result in the minimum cost.
  • Gradient Descent
    • It is an optimization algorithm used to find the values of parameters (coefficients) of a predictor function (h(x)) that minimizes the cost function (J (θ0, θ1) ).
    • This is used when the parameters can not be calculated using linear equations.
  • Classifier
    • It is a supervised algorithm which is used in classification of large volume of data to solve problems using machine learning.
    • It maps the input data to a specific category.
    • Few classifiers are:
      • Perceptron
      • Naive Bayes
      • Decision Tree
      • Logistic Regression
      • K-Nearest Neighbor
      • Artificial Neural Networks/Deep Learning
      • Support Vector Machine
  • Confusion Matrix
    • It is a technique to measure the performance of a classifier algorithm.
    • It gives the count as well as the types of error being made by a classifier algorithm.
    • A table is created with columns as the actual class and rows as the predicted class
      Confusion matrixPredicted Class
      OnOff
      Actual ClassOn428
      Off1832
    • The above table says
      • The classifier correctly identified On as On in 42 cases.
      • The classifier incorrectly identified On as Off in 8 cases.
      • The classifier correctly identified Off as Off in 32 cases
      • The classifier incorrectly identified Off as On in 18 cases
  • Classification Accuracy:
    • It is the ratio of correct predictions to total predictions.
      classification accuracy = correct predictions / total predictions
    • Misclassification/error rate:
      misclassification rate = (1 - classification accuracy) * 100
    • For the table mentioned in confusion matrix section
      classification accuracy = (42 + 32) / (42 + 8 + 32 + 18) = 74/100 = 0.74
      misclassification/error rate = (1 - 0.74 ) * 100 = 26
  • Sigmoid Function
    • It is a mathematical function having S shaped curve
    • Mathematically it is represented as
      S(x) = 1 / (1 + e x)
    • The output boundaries of this function are (0,1)

Categories of Machine Learning Algorithms

  • Supervised learning algorithm: The program is "trained" on a pre-defined set of "training examples", which then facilitate its ability to reach an accurate conclusion when given new data. The training data for the algorithms is already labeled. Below are the two types of supervised learning algorithms:
    • Classification Algorithms:
      • These algorithms give an output having a discrete label. They are used in case a given problem has a solution within a finite set of possible outcomes. They can be further of below types
        • Binary Classification: In case a problem can have only two possible outcomes e.g. determining whether an email is spam or not, if a candidate in an exam is a male or female.
        • Multi-label Classification: In case a problem can have more than two possible outcomes e.g. customer segmentation, audio and image categorization, and text analysis for mining customer sentiment.
      • Few examples of classification algorithms are Naive Bayes, decision trees, logistic regression, kernel approximation, and K-nearest neighbors.
    • Regression Algorithms:
      • These algorithms give an output which is unbounded. The outcome to a problem is represented by a quantity that is flexibly determined based on the inputs of the model rather than being confined to a set of possible labels e.g. predicting housing prices, likelihood of customers to churn, or the revenue a customer will generate.
      • Few examples of regression algorithms are linear regression, regression trees and support vector regression.
  • Un-supervised learning algorithm: The program is given a bunch of data and it itself must find patterns and relationships therein.
  • Semi-supervised learning algorithm:
  • Reinforcement learning (RL) algorithm:
    • The algorithm is based on interaction with environment.
    • The program dynamically interacts with the environment and improves its performance on the basis of positive and negative feedback it receives.
    • This involves the following
      • Agent: who performs all the tasks.
      • Action: Task performed by the agent.
      • Environment: Scenario faced by the agent. It is a modeled as a stochastic finite state machine with inputs (actions sent from the agent) and outputs (observations and rewards sent to the agent)
      • Reward function: Function used to train the agent how to perform the task.
      • Internal state State maintained by the agent to learn about the environment.
      • Policy: Strategy of choosing an action given a state in expectation of a better reward.
    • Some platforms associated with Reinforcement learning are:
      • Deepmind Lab
      • OpenAI Gym:
        • It is a toolkit for developing and comparing reinforcement learning algorithms.
        • This is the gym open-source library, which gives you access to a standardized set of environments.
      • OpenAI Universe
      • Project Malmo
      • VizDoom
      • RL-Glue
      • CommAI
      • Burlap
      • rlenvs
    • Steps involved in Reinforcement learning are
      • Observation of the environment
      • Deciding how to act using some strategy
      • Acting accordingly
      • Receiving a reward or penalty
      • Learning from the experiences and refining the strategy
      • Iterate until an optimal strategy is found
    • Few examples of Reinforcement learning are driverless cars, self navigating vaccum cleaners, scheduling of elevators etc.
