Create Reinforcement Learning Environment

Features

  • Creating an Environment:
    • A custom environment for reinforcement learning can be created using gym module of OpenAI.
    • gym module can be installed using pip unstall --user gym
    • Create a folder structure as below:
      • ReinforcementLearning
        • RLEnvironment
          • env
            • __init__.py
            • myenvironment.py
          • __init__.py
        • setup.py
    • setup.py is present inside the parent folder ReinforcementLearning and it sets up the name and version for the new environment. The content is as below

      from setuptools import setup
      setup(name='myenv', version='1.0', install_requires=['gym'])

      • __init__.py present inside the main module RLEnvironment registers the new environment. The content is as below

        from gym.envs.registration import register

        register(
        id='myenv-v0',
        entry_point='RLEnvironment.envs:MyEnvironment',
        )
      • __init__.py present inside the main module RLEnvironment/envs imports all the classes. The content is as below

        from RLEnvironment.envs.myenvironment import MyEnvironment

    • myenvironment.pyinside the folder ReinforcementLearning/RLEnvironment/envs consists of the actual environment class say MyEnvironment implementing gym.Env. This class needs to define four functions
      • __init__(self)
      • step(self, action)
      • reset(self)
      • render(self, ode='human',close=False)
  • Installing an Environment:
    • Using command prompt go to the parent folder ReinforcementLearning and install the above environment using command pip install --user -e .
  • Importing and using an Environment:
    • The environment can be imported inside another module using command import {moduleName} e.g. import RLEnvironment.envs.myenvironment
    • An instance of the environment imported above can be created as env = gym.make('myenv-v0')
  • Un-installing an Environment:
    • The environment can be un-installed using command pip uninstall {moduleName}
