Git Basics

Features

  • Forking: A fork is a copy of a repository. By forking a repository we can experiment with our changes/fixes without affecting the original project.
  • Adding SSH key on git hub: Run below commands in git bash

    • git config --global user.name "username"
    • git config --global user.email "email"
    • ssh-keygen -t rsa -b 4096 -C "email"
    • eval $(ssh-agent -s)
    • ssh-add ~/.ssh/id_rsa
    • clip < ~/.ssh/id_rsa.pub
    • Add the above copied text to git hub account settings under SSH and GPG keys
  • Amending the author for a commit : Run below commands in git bash

    • git config --global user.name "new username"
    • git config --global user.email "new email"
    • git commit --amend --reset-author
