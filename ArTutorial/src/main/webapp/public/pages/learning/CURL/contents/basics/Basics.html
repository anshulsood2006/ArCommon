CURL

Features

  • It stands for Client URL Request Library.
  • It is a command line tool for receiving and sending files using URL syntax.
  • It consists of different cURL Commands and libraries which can work with different protocols.

What is the difference between CURL and WGET?


CURL can be used to download multiples files in one go where as WGET can be used to download only a single file.
