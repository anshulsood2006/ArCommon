Python

Features

  • It is a programming language.
  • It was created in 1991 by Guido van Rossum.
  • It is used for
    • software development.
    • web development.
    • mathematics.
    • system scripting.
  • It can connect to database systems.
  • It can be used to handle big data and perform complex mathematics.
  • It works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc).
  • It has a simple syntax similar to the English language.
  • It is an interpreted language so the code can be executed as soon as it is written.
  • It can be used as an object-orientated language or functional language.
  • It uses new lines to complete a command instead of semicolon viz. ";" used in most of the programming languages.
  • It relies on indentation, using whitespace, to define scope; such as the scope of loops, functions and classes.
  • Its interpreter is easily extended with new functions and data types implemented in C or C++ (or other languages callable from C)
  • It allows to split a program into modules that can be reused in other Python programs.
  • It comes with a large collection of standard module like file I/O, system calls, sockets, and even interfaces to graphical user interface toolkits like Tk.
  • It enables programs to be written compactly and readably. Programs are shorter because:
    • It has high-level data types to express complex operations in a single statement.
    • The statements are grouped using indentation instead of braces.
    • The variables need not to be declared.
    • They can even change type after they have been set.

Some basic things related to python

  • After installation, add the installation directory into the path environmental variable to access it from any directory.
  • To check the version of python, use command python --version
  • Use ctrl+z or quit() or exit() to exit out of python.
  • If indentation is not given, python gives an IndentationError: expected an indented block
  • In case of invalid syntax, python gives an SyntaxError: invalid syntax
  • In case we try to add int to string, python gives an error TypeError: can only concatenate str (not "int") to str
  • To find the type of a variable use function type
  • Python Comments start with #
  • Python Docstring Python uses triple quotes at the beginning and end of a string to provide documentation.
  • Python VariablesThere is no need to declare a variable. A variable is created when we assign a value to it. Variable names:
    • must start with a letter or the underscore character
    • cannot start with a number
    • can only contain alpha-numeric characters and underscores
    • is case-sensitive
  • Python Numbers are of 3 types:
    • int e.g. x =2
    • float e.g. x =2.67
    • complex e.g. x =2j They are written with a "j" as the imaginary part
  • Python Casting It is used to specify the type of a variable using below constructor functions
    • int()
    • float()
    • str()
    • complex()
  • Python String It can be created using single or double quotes.
    • The strings in Python are arrays of bytes representing unicode characters.
    • Python does not have a character data type, a single character is simply a string with a length of 1.
    • str[n] is used to access nth elements of a string str.
    • str[n:m] is used to access elements from index n to m in a string str. Elements at index n and m both are not included.
    • str.strip() method is used to remove whitespace from start and end of a string str
    • len(str) method is used to return the length of a string str
    • str.lower() method is used to return a string str in lowercase.
    • str.upper() method is used to return a string str in uppercase.
    • str.replace("Z","K") method is used to replace Z in a string str with K.
    • str.split(separator) method is used to split a string str using given separator.
    • input() method is used for command line input.
