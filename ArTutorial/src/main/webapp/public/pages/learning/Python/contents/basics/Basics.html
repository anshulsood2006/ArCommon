Python

Features

  • It is a programming language.
  • It was created in 1991 by Guido van Rossum.
  • It is used for
    • software development.
    • web development.
    • mathematics.
    • system scripting.
  • It can connect to database systems.
  • It can be used to handle big data and perform complex mathematics.
  • It works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc).
  • It has a simple syntax similar to the English language.
  • It is an interpreted language so the code can be executed as soon as it is written.
  • It can be used as an object-orientated language or functional language.
  • It uses new lines to complete a command instead of semicolon viz. ";" used in most of the programming languages.
  • It relies on indentation, using whitespace, to define scope; such as the scope of loops, functions and classes.
  • Its interpreter is easily extended with new functions and data types implemented in C or C++ (or other languages callable from C)
  • It allows to split a program into modules that can be reused in other Python programs.
  • It comes with a large collection of standard module like file I/O, system calls, sockets, and even interfaces to graphical user interface toolkits like Tk.
  • It enables programs to be written compactly and readably. Programs are shorter because:
    • It has high-level data types to express complex operations in a single statement.
    • The statements are grouped using indentation instead of braces.
    • The variables need not to be declared.
    • They can even change type after they have been set.

  • Python Emailing
    • Simple Mail Transfer Protocol (SMTP) is a protocol, which handles sending e-mail and routing e-mail between mail servers.
    • Python uses smtplib module for sending emails.

      import smtplib

    • Open smtp session as session = smtplib.SMTP('smtphost',smtpport).
    • Use smtp session to send message as session.sendmail(sender, receivers, message).
    • Quit smtp session as session.quit()
    • Example:

      import smtplib
      from smtplib import SMTPException

      sender = 'sender@gmail.com'
      receiver = ['receiver@gmail.com']
      message = """
      Subject: SMTP e-mail test
      This is a test e-mail message sent via Python script.
      """

      try:
           smtpObj = smtplib.SMTP('localhost')
           smtpObj.sendmail(sender, receivers, message)
           smtpObj.quit()
      except SMTPException as e:
           print("Error: unable to send email",e)

  • Python Multithreading
    • Using Thread Module
      • Import the thread module as import _thread.
      • Create a function func(arguments) to be executed inside thread. Make sure that there is a return statement.
      • Create thread as _thread.start_new_thread(func, (arguments)).
      • Example:
        import _thread
        import time

        def threadFunction(param1, param2):
             print(param1)
             print(param2)
             return

        try:
             _thread.start_new_thread(threadFunction, ("Anshul","Sood"))
             _thread.start_new_thread(threadFunction, ("Karan","Ahuja"))
             time.sleep(1)
        except Exception as e:
             print("Exception in thread received", e)

    • Using Threading Module
      • Import the threading module as import threading
      • Create a subclass of Thread class as class myThreading(threading.Thread):
      • Override the __init__(self [,args]) method to add additional arguments. Make sure that the first statement inside the __init__ method is threading.Thread.__init__(self)
      • Override the run(self [,args]) method to implement the logic that thread needs to execute. To get the additional arguments inside run method use self.argument
      • Create thread using thread = myThreadig(arguments)
      • Start thread using thread.start()
      • Example:
        import threading
        import time

        class myThreading(threading.Thread):

             def __init__(self, param1, param2):
                 threading.Thread.__init__(self)
                 self.param1 = param1
                 self.param2 = param2

             def run(self):
                 print(self.param1)
                 print(self.param2)
                 return

        try:
             thread1 = myThreading("Anshul","Sood")
             thread2 = myThreading("Karan","Ahuja")
             thread1.start()
             thread2.start()
             time.sleep(1)
        except Exception as e:
             print("Exception in thread received", e)

  • Python XML Parsing
    • SAX Parsing
      • Import the sax parser module as import xml.sax
      • Create a content handler class as subclass of xml.sax.ContentHandler class
      • Override the __init__(self) method to initialize all the xml tags content tp empty string "". Also set a variable say current data to "". This will hold the current tag while parsing.
      • Override the startElement(self, tag, attributes) method. This method is called when an element tag is started
        • Set the current data attribute to tag
        • All the attributes associated with the tag can be retrieved using method attributes["attributeName"]
      • Override the endElement(self, tag) method. This method is called when an element tag ends.
      • Override the characters(self, content) method. This method is called when a character is read.
        • Set all the attributes to content as self.attributeName = content based on current data attribute. e.g.
          if self.CurrentData == "elementName":
               self.type = content
      • Create an XML parser as parser = xml.sax.make_parser()
      • Turn off the namespaces as parser.setFeature(xml.sax.handler.feature_namespaces, 0)
      • Set the content handler as parser.setContentHandler(ContentHandlerClass())
      • Parse the given XML using appropriate protocol e.g. file: or http: and using the parser as parser.paser(xmlFile)
      • Example:
        import xml.sax
        import os

        class MyContentHandler(xml.sax.ContentHandler):
             def __init__(self):
                 self.CurrentData = ""
                 self.name = ""
                 self.age = 0
                 self.gender = ""
                 return

             def startElement(self, tag, attributes):
                 self.CurrentData = tag
                 if (tag == "member"):
                     print("****Member****")
                 return

             def endElement(self, tag):
                 if (self.CurrentData == "name"):
                     print("Name = ",self.name)
                 if (self.CurrentData == "age"):
                     print("Age = ",self.age)
                 if (self.CurrentData == "gender"):
                     print("Gender = ",self.gender)
                 self.CurrentData = ""
                 return

             def characters(self, content):
                 if (self.CurrentData == "name"):
                     self.name = content
                 if (self.CurrentData == "age"):
                     self.age = content
                 if (self.CurrentData == "gender"):
                     self.gender = content
                 return

        parser = xml.sax.make_parser()
        parser.setFeature(xml.sax.handler.feature_namespaces, 0)
        parser.setContentHandler(MyContentHandler())
        path = "file:///"+os.getcwd()+"/family.xml"
        parser.parse(path)

  • Python Networking
    • Python provides two level of access to network services
      • Operating system level socket to implement clients and servers for both connection-oriented and connection less protocols.
      • Application-level network protocols, such as FTP, HTTP, and so on
    • Sockets
      • These are the end points of a bidirectional communications channel.
      • They may communicate within a process, between processes on the same machine, or between processes on different continents.
      • They can be implemented over a number of different channel types: Unix domain sockets, TCP, UDP, and so on.
    • To create a server socket
      • Import the module socket as import socket
      • Create socket object as server = socket.socket()
      • Get the host as host = socket.gethostname()
      • Set some random port as port = 12345
      • Bind the server socket object to the host and port as server.bind((host, port)). See there is only one argument inside parenthesis as (host,port)
      • Wait for client connection as server.listen(5)
      • Start an infinite loop to listen for infinite requests as while True: server.accept(). This method when receives a message from client returns connection object and IP address of the client machine.
      • To write something back to the client use as connection.send(b'String to send'). Send method accepts byte so use b'str'.
      • Example

        import socket

        serverSocket = socket.socket()
        host = socket.gethostname()
        port = 12345
        serverSocket.bind((host, port))
        serverSocket.listen(5)
        while True:
             print("Waiting for connection from client")
             connection, addr = serverSocket.accept()
             print("Got Connection from ", addr)
             connection.sendall(b'Thanks for connecting.')
             connection.close()

    • To create a client socket
      • Import the module socket as import socket
      • Create client socket object as server = clientSocket.socket()
      • Get the host as host = socket.gethostname()
      • Connect to the server socket as clientSocket.connect((serverHost, serverPort))
      • Write to the socket as clientSocket.send(b'str')
      • Close the client socket as clientSocket.close()
      • Example

        import socket

        clientSocket = socket.socket()
        host = socket.gethostname()
        port = 12345
        clientSocket.connect((host, port))
        clientSocket.recv(1024)
        clientSocket.close()
