Data Structures

Features

The data structures present in java.util package are:

  • Enumeration: It is a legacy interface to iterate over a collection of objects. This is obtained by
    • For legacy data structures, call elements() methods on the object.
    • For collections, call Collections.enumeration(collectionObject)
    	public static void main(String[] args) {
		List list = null;
		list = new ArrayList<>();
		list.add("Anshul");
		list.add("Akhil");
		list.add("Abhishek");
		list.add("Ashok");
		Enumeration e = Collections.enumeration(list);
		while (e.hasMoreElements()){
			System.out.println(e.nextElement());
		}
		Vector vector = new Vector<>();
		vector.add("Anshul");
		vector.add("Akhil");
		vector.add("Abhishek");
		vector.add("Ashok");
		e = vector.elements();
		while (e.hasMoreElements()){
			System.out.println(e.nextElement());
		}
	}
  • BitSet: Special type of array that holds bit values.
  • Vector: Synchronized list of objects having some additional legacy methods.
  • Stack: Stack is a subclass of Vector that implements a standard last-in, first-out stack.
  • Dictionary: Legacy class to store elements in key value pair.
  • Hashtable: Synchronized legacy class to store elements in key value pair. Earlier it was concrete implementation of a Dictionary, now it implements Map interface as well.
  • Properties: Properties is a subclass of Hashtable and is used to maintain lists of values in which the key is a String and the value is also a String.
All the above classes are now legacy and Java-2 has introduced a new framework called Collections Framework.
