Sleep()

Features

It is used to sleep the thread for a specified amount of time.

It has two variants

  • sleep(long miliseconds)
  • sleep(long miliseconds, int nanoseconds)

Both the methods can throw InterruptedException

Examples

sleep() Method

public class ArSleepExample implements Runnable{
	
	private String name;
	
	public ArSleepExample(String name){
		this.name = name;
	}

	@Override
	public void run() {
		System.out.println("Thread: "+name +" is waiting for 5000 milseconds");	
		try {
			Thread.sleep(1000);
		} catch (InterruptedException e) {
			e.printStackTrace();
		}
	}
	
	public static void main(String[] args) {
		Thread t1 = new Thread(new ArSleepExample("Anshul"));
		Thread t2 = new Thread (new ArSleepExample("Akhil"));
		t1.start();
		t2.start();
	}
}
