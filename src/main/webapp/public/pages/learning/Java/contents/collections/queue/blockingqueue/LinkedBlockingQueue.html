Linked Blocking Queue

Features

  • It uses nodes (like a linked list), to keep track of the order of the elements, which increases the complexity of the data structure. It can have a fixed-size limit as well, but if we don't define one the limit is Integer.MAX_VALUE by default.
  • Linked queues typically have higher throughput than array-based queues but less predictable performance in most concurrent applications.
