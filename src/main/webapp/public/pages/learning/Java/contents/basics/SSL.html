SSL

Features

  • Keystore: It is used to store server's own identity/certificates. It contains the server's certificate, including its private key.
  • Truststore: It is used to store identity/certificates of other parties issued by CA
  • JSSE: Java Secure Socket Extension

Examples

Create a SSL connection between web server and the client.

  • Generate certificate:

    keytool -genkey -alias localhost -keyalg RSA -keypass password -storepass password -keystore server.keystore.jks

  • Convert to cer file:

    keytool -export -alias localhost -storepass password -file server.cer -keystore server.keystore.jks

  • Add the certificate to the trust store file:

    keytool -import -v -trustcacerts -alias localhost -file server.cer -keystore server.truststore.jks -keypass password -storepass password

  • Verify if the certificate was successfully added into the keystore.

    keytool -list -v -keystore server.keystore.jks

  • Validate if the certificate was successfully added into the trust store.

    keytool -list -v -keystore server.truststore.jks

  • Delete a certificate file:

    keytool -delete -alias sb6270x1769 -keystore server.keystore.jks
